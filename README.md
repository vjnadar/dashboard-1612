**Introduction**

A neat looking **Dashboard app template** built using **React JS and Typescript**! 

**Key features**
- It's a Dashboard app template. Ninety percent of the application was built using plain **HTML and Sass**!   
- It uses **React JS** and **Typescript**.
- It's fully **responsive** and uses **CSS grid**.
- It uses **Jest** and the **React Testing Library** for testing.
- It uses **Webpack 5**.

## Installation

 **Frontend**:
 
1. Copy this repository and enter the **project** directory using the terminal.
2. Use **npm install** to install the necessary node modules.
4. Use **npm start** to start up the frontend server.
